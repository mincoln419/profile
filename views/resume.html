국민연금외화금고 구축 프로젝트

2018.03. ~ 2018.08.
- 수행사: 코어뱅크
- 고객사: 하나은행
내용
국민연금 외화자금 투자, 관리를 위한 금고 시스템 구축.
하나은행 계정계 시스템, 국민연금과 Interface를 통해 국민연금 내부망에서 직접 외화투자관리가 가능하도록함.
담당업무
업무공통파트.
외화금고 시스템의 공통 화면, 팝업 및 모듈 프로그램 개발.
국제 금융기관에서 공통으로 사용하는 기초데이터 수신 인터페이스 개발.
하나은행 계정계 시스템으로 부터 정기적으로 코드성 데이터, 환율데이터를 자동 최신화 시키는 배치프로그램 개발.
개발환경
  • Java Project
  • OS: UNIX
  • WAS: ZEUS
  • Framwork: SPRING(v 3.2), Yframe(코어뱅크 자체 js framework), ibatis
  • Sceduler: ControlM
  • DB tool: Orange
